Tech Career Paths

What They Do & Skills You Will Learn

Software Developer / Software Engineer

Develop applications, websites, and software solutions. Write, test, and maintain code.

Skills: Python, JavaScript, Java, C#, Git, Agile

Pathways: University, Apprenticeships, Bootcamps

Front-End Developer

Build the visual and interactive parts of websites and applications.

Skills: HTML, CSS, JavaScript, React, UX/UI

Pathways: University, Bootcamps, Apprenticeships

Back-End Developer

Work on the behind-the-scenes functionality of websites, managing servers, databases, and APIs.

Skills: Node.js, Python, Java, Databases, APIs

Pathways: University, Bootcamps, Apprenticeships

Cyber Security Analyst

Protect organisations from cyber threats and conduct security audits.

Skills: Network security, Penetration testing, Cryptography

Pathways: University, Certifications, Apprenticeships

Cloud Engineer

Build and maintain cloud infrastructure for scalable applications.

Skills: AWS, Azure, Google Cloud, Kubernetes, Terraform

Pathways: Certifications, Bootcamps, Work Experience

Blockchain Developer

Develop decentralized applications (DApps) and smart contracts.

Skills: Solidity, Ethereum, Smart Contracts, Cryptography

Pathways: Self-study, Certifications, Online Courses

Embedded Systems Engineer

Develop software for hardware devices like IoT and microcontrollers.

Skills: C, C++, Microcontrollers, Real-time OS

Pathways: University, Certifications, Work Experience

Game Developer

Create interactive video games for various platforms.

Skills: Unity, Unreal Engine, C#, Game Physics

Pathways: University, Online Courses, Game Jams

UI/UX Designer

Design intuitive user interfaces and user experiences.

Skills: Figma, Adobe XD, User Research, Wireframing

Pathways: Bootcamps, Certifications, Online Courses

Product Manager

Oversee product development and ensure market fit.

Skills: Agile, Market Research, Roadmaps, Business Strategy

Pathways: MBA, Work Experience, Certifications
